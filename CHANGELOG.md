# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-24

### Added
- Initial project structure with core functionality
- CLI tool via `bin/studio.js`
- Legacy build system for NW.js applications
- Support for multiple platforms (macOS, Windows, Linux)
- Package management and dependency handling
- App signing capabilities
- DMG creation for macOS distributions

### Changed
- Migrated from legacy codebase to modern ES modules
- Updated dependencies to latest stable versions

### Fixed
- Various compatibility issues with modern Node.js versions

## [1.0.0] - Initial Release

### Added
- Core Studio functionality for NW.js application development
- Build, pack, and run commands
- Installation and uninstallation utilities
- Configuration management
- Sample project generation
- Synchronization capabilities
