# Handle open-with

Procedures pre installed:

1. windows
	right-click -> open with -> choose another app -> more apps [nw.exe]

Posibility open-with:

- single file html <- [current]
- single file non-html
- folder
- multi file
- zip
- and more ..

## Create handler

In windows, manualy, create `ink` file

- create new shortcut
- point to `full-path\to\nw.exe full-path\to\target\`, give name `my_app` 
- path\to\target **should not** contain space, path with "" not work
- test open-with `file.html` with `my_app.ink`
- it can also just **drag** on to it
- it should point to folder `example`
